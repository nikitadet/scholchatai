# scholchatai
для начала работы нужен ключ. гайд:
1. перейти на сайт Mistral AI и создать учётную запись. Для этого нужно перейти по адресу: mistral.ai и нажать на кнопку «Войти» или «Зарегистрироваться».
2. Выбрать план. Можно подписаться на бесплатный план Experiment или на план «Плати по мере использования». План Scale имеет более высокие лимиты, но для него нужно заполнить информацию об оплате. При подписании плана потребуется подтвердить номер телефона. поддерживает Российские номера
3. Создать новый API-ключ. На левой панели навигации нужно нажать на «API-ключи» и найти большую оранжевую кнопку «Создать новый ключ».
4. Скопировать ключ. После генерации API-ключ нельзя будет увидеть снова, его нужно сохранить в безопасном месте и не делиться с другими.
